## Projects
### 1. TaskNest – To-Do List Web Application
TaskNest is a responsive task management web application built using HTML, CSS, and JavaScript. It helps users organize daily activities through an intuitive interface and browser-based data storage.

#### Features
* Add new tasks
* Mark tasks as completed
* Delete individual tasks
* Delete all tasks
* Mark all tasks as completed
* Active and completed task statistics
* LocalStorage support for persistent data
* Responsive design for desktop and mobile devices

### 2. GritHouse – Modern Gym Website

GritHouse is a modern, responsive single-page gym website built using HTML5, CSS3, Bootstrap 5, and JavaScript. It features a professional dark-themed interface, smooth animations, interactive sections, and a mobile-friendly design tailored for a men's fitness club.

#### Features

- Modern responsive design
- Sticky navigation bar
- Smooth scrolling
- Animated statistics counters
- Hero section with call-to-action buttons
- Featured training programs
- Premium facilities showcase
- Why Choose Us section
- Membership plans
- Contact form
- Back-to-top button
- Interactive hover effects
- Professional footer
