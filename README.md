# Hex Software Internship Projects
This repository contains projects developed during my Web Development Internship at Hex Software. Each project demonstrates practical frontend development skills, responsive design principles, and JavaScript-based functionality.

## Projects
### 1. TaskNest – To-Do List Web Application
TaskNest is a responsive task management web application built using HTML, CSS, and JavaScript. It helps users organize daily activities through an intuitive interface and browser-based data storage.

#### Features
* Add new tasks
* Mark tasks as completed
* Delete individual tasks
* Delete all tasks
* Mark all tasks as completed
* Active and completed task statistics
* LocalStorage support for persistent data
* Responsive design for desktop and mobile devices
