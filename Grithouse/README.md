# GritHouse – Modern Gym Website

GritHouse is a modern, responsive single-page gym website developed using HTML5, CSS3, Bootstrap 5, and JavaScript. The website is designed specifically for a men's fitness club and provides a professional user experience with modern UI design, smooth animations, responsive layouts, and interactive elements. It showcases essential sections commonly found on commercial gym websites, including programs, facilities, membership plans, and contact information.

## Features

* Fully responsive single-page website
* Modern and professional dark-themed UI
* Sticky navigation bar
* Smooth scrolling navigation
* Animated statistics counters
* Interactive hover effects
* Hero section with call-to-action buttons
* About GritHouse section
* Featured training programs
* Premium facilities showcase
* Why Choose Us section
* Membership pricing plans
* Contact form
* Social media integration
* Back-to-top button
* Professional footer

## Project Objective

The objective of GritHouse is to design and develop a modern gym landing page that demonstrates front-end web development skills, responsive design principles, user interface design, and JavaScript-based interactivity. The project focuses on creating an engaging and visually appealing experience suitable for a commercial fitness website.

## Website Sections

### Hero Section

Features an engaging landing area with a motivational headline, call-to-action buttons, and highlights of the gym's services.

### About GritHouse

Introduces the gym's mission, values, and commitment to helping members achieve their fitness goals.

### Animated Statistics

Displays key gym statistics with animated counting effects for an engaging user experience.

### Featured Programs

Showcases the different fitness programs offered, including Strength Training, Bodybuilding, Boxing, CrossFit, HIIT, and Cardio Fitness.

### Facilities

Highlights the gym's premium facilities using modern image cards.

### Why Choose GritHouse

Presents the key benefits of joining GritHouse, including certified trainers, modern equipment, flexible schedules, and a safe training environment.

### Membership Plans

Displays multiple membership packages with pricing and included benefits.

### Contact Section

Provides contact information, social media links, and a responsive contact form.

### Footer

Contains quick navigation links, contact details, and copyright information.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open the `index.html` file in any modern web browser.
4. Explore the website and its interactive features.
