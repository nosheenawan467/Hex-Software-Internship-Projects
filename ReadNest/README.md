# ReadNest – Modern Digital Library Website

ReadNest is a responsive digital library web application built using HTML, CSS and JavaScript. It allows users to browse books, search titles, borrow and return books, manage a wishlist and track reading activity using LocalStorage.

## Features

- Responsive modern UI
- Hero image slider
- Sticky navigation
- Book search
- Book categories
- Recently Added books slider
- Borrow / Return books
- Borrow History
- Wishlist
- Reading statistics
- LocalStorage support
- Mobile friendly design
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Swiper.js
- Font Awesome

## Main Functionalities

### Search
Search books by title with live filtering.

### Borrow System
Borrow and return books from the library.

### Borrow History
Automatically stores borrowing records using LocalStorage.

### Wishlist
Save favorite books for later reading.

### Reading Statistics
Displays:
- Total Books
- Currently Reading
- Borrow History
- Wishlist

### Responsive Design
Optimized for desktop, tablet and mobile devices.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open the ReadNest folder.
4. Open the index.html file in any modern web browser.
5. Explore the digital library and its interactive features.
