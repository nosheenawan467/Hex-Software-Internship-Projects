# TaskNest – To-Do List Web Application

TaskNest is a responsive and user-friendly To-Do List web application developed using HTML, CSS, and JavaScript. The application helps users organize daily activities efficiently by allowing them to create, manage, track, and complete tasks. All task data is stored locally in the browser using the LocalStorage API, ensuring persistence even after page refreshes.

## Features
* Add new tasks
* Mark tasks as completed
* Delete individual tasks
* Delete all tasks at once
* Mark all tasks as completed
* Display active and completed task counts
* Store tasks using browser LocalStorage
* Fully responsive design for desktop and mobile devices
* Multi-page navigation (Home, Add Task, View Tasks)
* Interactive hover effects and modern UI styling

## Project Objective
The objective of TaskNest is to provide a simple and effective task management solution while demonstrating fundamental web development concepts such as DOM manipulation, event handling, responsive design, and browser-based data storage.

## Application Pages
### Home Page
Provides navigation to the Add Task and View Tasks pages.

### Add Task Page
Allows users to create new tasks and save them to LocalStorage.

### View Tasks Page
Displays all saved tasks with options to:
* Mark tasks as completed
* Delete individual tasks
* Mark all tasks as completed
* Delete all tasks
* View task statistics
